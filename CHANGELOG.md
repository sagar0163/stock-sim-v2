# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-22

### Added
- Core backend with Node.js and Express.
- MongoDB models for Users, Stocks, Transactions, and Market Events.
- Authentication system with JWT.
- Market simulation service with real-time updates via Socket.io.
- Portfolio management (Buy/Sell logic).
- Leaderboard system.
- Watchlist functionality.
- Vue.js 3 frontend with a responsive dashboard.
- Real-time market price feed in the UI.
- Project specification and API documentation.
- Docker and Docker-compose support.
- Initial seed data for 20 major stocks.
